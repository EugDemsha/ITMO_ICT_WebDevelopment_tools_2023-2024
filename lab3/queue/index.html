    Вызов парсера через очередь

    Задача: Необходимо добавить зависимости для Celery и Redis в проект. Celery будет использоваться для обработки задач в фоне, а Redis будет выступать в роли брокера задач и хранилища результатов.

    Приложение Celery и задача для него:

    from celery import Celery
from parser.parse_utils import parse, save

celery_app = Celery(
    "tasks", broker="redis://redis:6379/0", backend="redis://redis:6379/0"
)

celery_app.conf.update(
    task_serializer="json",
    accept_content=["json"],
    result_serializer="json",
    timezone="UTC",
    enable_utc=True,
)

@celery_app.task
def celery_parse_and_save(url):
    data = parse(url)
    save(data)
    print(f"Saved currency: {data['name']}")

    Файл main.py приложения парсера

    from fastapi import FastAPI
import uvicorn
from parser.threading_parser import crypto_parse
from parser.settings import URLS

app = FastAPI()


@app.post("/celery_parse_cryptocurrencies/num")
def parse_async(num: int):
    urls_slice = URLS[:num]
    for url in urls_slice:
        celery_parse_and_save.delay(url)
    return {"message": "Parsing started"}
